# FilmReader 依赖包列表
# 
# 安装方法：
#   pip install -r requirements.txt
#
# 核心依赖（必需）:
#   - Pillow: 图像处理
#   - mss: 屏幕捕获（推荐，性能最好）
#   - edge-tts: Edge TTS 语音合成（免费，推荐）
#   - pygame: 音频播放
#
# OCR 依赖（至少选择一个）:
#   - pytesseract: Tesseract OCR 包装器（需要额外安装 Tesseract）
#
# 可选依赖：
#   - pydub: 音频处理（如需使用 pydub 播放器）
#   - pyautogui: 屏幕捕获（备选方案）
#   - numpy: 数组处理（某些 OCR 需要）
#   - opencv-python: 图像处理（高级功能）
